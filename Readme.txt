Meu código para um carrinho de compras em JavaScript usando elementos do DOM e uma API REST, usando práticas do JSON e manipulação. Este foi um exercício da faculdade, era o último módulo de manipulação do JSON usando todas as técnicas e aprendizados do módulo de JavaScript. Para mim, foi um desafio, mas consegui concluir. Foi um projeto de adicionar itens, somar preços, somar total, mudar elementos do DOM, consumir uma API REST pública. Faculdade Descomplica Digital. O HTML é um template, já o código em si foi criado na prática do módulo.
